1. Vector Spaces and Subspaces: The Four Fundamental Subspaces and Rank-Nullity TheoremMathematical BackgroundA matrix A∈Rm×nA \in \mathbb{R}^{m \times n}A \in \mathbb{R}^{m \times n}
 represents a linear map A:Rn→RmA: \mathbb{R}^n \to \mathbb{R}^mA: \mathbb{R}^n \to \mathbb{R}^m
. The four fundamental subspaces are:Column space (range): C(A)={Ax∣x∈Rn}⊆Rm\mathcal{C}(A) = \{ Ax \mid x \in \mathbb{R}^n \} \subseteq \mathbb{R}^m\mathcal{C}(A) = \{ Ax \mid x \in \mathbb{R}^n \} \subseteq \mathbb{R}^m
, dim = rank(A).
Null space (kernel): N(A)={x∈Rn∣Ax=0}\mathcal{N}(A) = \{ x \in \mathbb{R}^n \mid Ax = 0 \}\mathcal{N}(A) = \{ x \in \mathbb{R}^n \mid Ax = 0 \}
, dim = n - rank(A) (rank-nullity: rank(A) + nullity(A) = n).
Row space: C(AT)⊆Rn\mathcal{C}(A^T) \subseteq \mathbb{R}^n\mathcal{C}(A^T) \subseteq \mathbb{R}^n
.
Left null space: N(AT)⊆Rm\mathcal{N}(A^T) \subseteq \mathbb{R}^m\mathcal{N}(A^T) \subseteq \mathbb{R}^m
, dim = m - rank(A).

These relate via orthogonality: N(A)⊥C(AT)\mathcal{N}(A) \perp \mathcal{C}(A^T)\mathcal{N}(A) \perp \mathcal{C}(A^T)
, N(AT)⊥C(A)\mathcal{N}(A^T) \perp \mathcal{C}(A)\mathcal{N}(A^T) \perp \mathcal{C}(A)
.Numerically, we compute them using SVD: A=UΣVTA = U \Sigma V^TA = U \Sigma V^T
, where columns of U and V provide bases.Computational ExampleLet's take a 2x3 matrix A and compute subspaces.python

# Define matrix A (2x3, rank 2)
A = np.array([[1, 2, 3], [4, 5, 6]]).T  # Transpose to make it 3x2 for better viz, but treat as 2x3
print("Matrix A:\n", A)

# SVD for subspaces
U, S, Vt = np.linalg.svd(A, full_matrices=True)
rank = np.sum(S > 1e-10)  # Numerical rank
print(f"Rank: {rank}")

# Column space: first rank columns of U
col_space_basis = U[:, :rank]
print("Column space basis:\n", col_space_basis)

# Null space: use scipy
null_basis = null_space(A)
print("Null space basis:\n", null_basis)

# Row space: first rank rows of Vt.T (V columns)
row_space_basis = Vt[:rank, :].T
print("Row space basis:\n", row_space_basis)

# Left null space: last m-rank columns of U
left_null_basis = U[:, rank:]
print("Left null space basis:\n", left_null_basis)

# Verify rank-nullity: rank + dim(null) = n
print(f"Rank-nullity check: {rank} + {null_basis.shape[1]} == {A.shape[1]}? {rank + null_basis.shape[1] == A.shape[1]}")

2. Linear Independence, Span, Basis, DimensionMathematical BackgroundVectors {v1,…,vk}\{v_1, \dots, v_k\}\{v_1, \dots, v_k\}
 are linearly independent if ∑αivi=0\sum \alpha_i v_i = 0\sum \alpha_i v_i = 0
 implies all αi=0\alpha_i = 0\alpha_i = 0
. They span a subspace if every vector in it is a linear combination. A basis is a maximal independent set that spans; dimension is the basis size.In NLA, we check independence via rank of matrix with vectors as columns: rank = k iff independent.Computational Examplepython

# Vectors in R^3

rank_V = np.linalg.matrix_rank(V)
print(f"Rank: {rank_V} (independent if ==3: {rank_V == 3})")

# Basis: reduce to independent set (e.g., via QR or manual)
print("A basis for span: first two vectors (independent)")

# Dimension: rank
dim = rank_V
print(f"Dimension of span: {dim}")

3. Linear Maps and Matrix RepresentationsMathematical BackgroundA linear map T:V→WT: V \to WT: V \to W
 has matrix representation [T] relative to bases of V and W. Changing basis: if P changes basis in domain, Q in codomain, new matrix is Q^{-1} A P.In NLA, matrices are maps; eigenvalues describe scaling in invariant directions.Computational Examplepython

# Linear map A: R^2 -> R^2
A_map = np.array([[2, 1], [1, 3]])

# Apply to standard basis
e1 = np.array([1,0])
Ae1 = A_map @ e1
print("A e1:", Ae1)

# Change basis: P = rotation by 45 deg
theta = np.pi/4
P = np.array([[np.cos(theta), -np.sin(theta)], [np.sin(theta), np.cos(theta)]])
A_new = np.linalg.inv(P) @ A_map @ P  # Same map, new basis
print("A in new basis:\n", A_new)

4. Inner Products and NormsMathematical BackgroundInner product: ⟨u,v⟩=uTv\langle u, v \rangle = u^T v\langle u, v \rangle = u^T v
 (Euclidean). Norm: ∥v∥2=⟨v,v⟩\|v\|_2 = \sqrt{\langle v,v \rangle}\|v\|_2 = \sqrt{\langle v,v \rangle}
.Matrix norms: induced ∥A∥p=sup⁡x≠0∥Ax∥p/∥x∥p\|A\|_p = \sup_{x \neq 0} \|Ax\|_p / \|x\|_p\|A\|_p = \sup_{x \neq 0} \|Ax\|_p / \|x\|_p
; Frobenius \|A\|_F = \sqrt{\sum a_{ij}^2} = \sqrt{\trace(A^T A)}.Orthogonality: ⟨u,v⟩=0\langle u,v \rangle = 0\langle u,v \rangle = 0
.In NLA, norms measure errors, conditioning.Computational Examplepython

# Vector norms
v = np.array([3,4])
norm2 = np.linalg.norm(v, 2)
print("2-norm:", norm2)

# Matrix norms
A_norm = np.array([[1,2],[3,4]])

norm_frob = np.linalg.norm(A_norm, 'fro')
print("Induced 2-norm:", norm_ind2)
print("Frobenius norm:", norm_frob)

# Orthogonality check
u = np.array([1,1])
v_orth = np.array([-1,1])

5. Projections: Orthogonal ProjectorsMathematical BackgroundFor orthonormal Q (columns), projector P = Q Q^T satisfies P^2 = P, P^T = P.Projects onto span(Q): Px = closest point in subspace.In NLA, used in least squares: argmin |b - Ax| = projection of b onto C(A).Computational Examplepython

# Orthonormal Q (2x2 identity for simplicity)
Q = np.eye(2)

# Projector
P = Q @ Q.T
print("P:\n", P)

# Properties
print("P^2 == P?", np.allclose(P @ P, P))
print("P^T == P?", np.allclose(P.T, P))

# Project a vector
x = np.array([3,4])
Px = P @ x
print("Projection:", Px)

Eigenvalues and Eigenvectors: Spectral Theorem for Symmetric MatricesMathematical BackgroundFor symmetric A = A^T, exists orthogonal Q such that A = Q \Lambda Q^T, with real \Lambda diagonal (eigenvalues), Q columns eigenvectors.Eigenvectors orthogonal, eigenvalues real.In NLA, used for diagonalization, PCA, stability.Computational Examplepython

# Symmetric matrix
A_sym = np.array([[2,1],[1,2]])

# Eigendecomposition
eigvals, eigvecs = np.linalg.eigh(A_sym)  # For symmetric
print("Eigenvalues:", eigvals)
print("Eigenvectors:\n", eigvecs)

# Check orthogonality: Q^T Q = I
print("Q^T Q == I?", np.allclose(eigvecs.T @ eigvecs, np.eye(2)))

# Reconstruct
Lambda = np.diag(eigvals)
A_recon = eigvecs @ Lambda @ eigvecs.T
print("Reconstructed A:\n", A_recon)

7. Rank: Deficiency and Numerical CatastrophesMathematical BackgroundRank(A) = dim(C(A)). Deficiency: rank < min(m,n).Near-deficiency: small singular values, high condition number \kappa(A) = \sigma_1 / \sigma_r \gg 1.Numerically catastrophic: ill-conditioned systems amplify errors.Computational Examplepython

# Full rank
A_full = np.array([[1,2],[3,4]])
rank_full = np.linalg.matrix_rank(A_full)
cond_full = np.linalg.cond(A_full)
print(f"Full: rank={rank_full}, cond={cond_full}")

# Deficient
A_def = np.array([[1,2],[2,4]])
rank_def = np.linalg.matrix_rank(A_def)
print(f"Deficient: rank={rank_def}")

# Near-deficient: perturb
eps = 1e-8
A_near = np.array([[1,2],[2,4 + eps]])
cond_near = np.linalg.cond(A_near)
print(f"Near-deficient cond: {cond_near}")

# Numerical issue: solve Ax=b with perturbation
b = np.array([5,9])
x = np.linalg.solve(A_full, b)
print("Solution full:", x)

# Perturbed b
b_pert = b + np.array([1e-6, 1e-6])
x_pert = np.linalg.solve(A_near, b_pert)
print("Perturbed near-def sol:", x_pert)  # Large change if cond high

VisualizationSingular values plot for near-deficiency.python

# SVD
_, S_full, _ = np.linalg.svd(A_full)
_, S_near, _ = np.linalg.svd(A_near)

fig, ax = plt.subplots()
ax.plot(S_full, 'bo-', label='Full')
ax.plot(S_near, 'ro-', label='Near-def')
ax.set_yscale('log')
ax.set_title('Singular Values (log scale)')
ax.legend()
plt.show()
